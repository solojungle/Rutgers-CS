** USE THIS PAGE TO ENTER SQL QUERIES THAT INSERT, UPDATE, OR DELETE ENTRIES IN THE DATABASE

** NOTE: ANY PRIMARY KEY THAT IS AN ID IS SET TO AUTO INCREMNT IN THE DATABASE,

** SO DO NOT INSERT ANY VALUE FOR THOSE COLUMNS!

** Any column that requires a time to be entered is in the format of a 24 Hour Clock, 'HH:MM:SS' (ex.'13:00:00')

** Any column that requires a date to be entered is in the format of 'YYYY-MM-DD' (ex. '2018-11-18')

** For the item table in the category column put:

  • 'beer' - for alcholic beverage (beers)
  • 'drink' - for non-alcholic beverages (ex. soda, water)
  • 'food' - for food items (ex. hot dog, hamburger)
